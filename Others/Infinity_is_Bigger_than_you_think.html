Infinity is the idea of being Endless

LISTABLE

Integars are definitly listable by just keep adding one to the last number in the list to reach infinity.
Eg. 1,2,3,4,5,6,7,8,9,...


But, are fractions listable? Are we able to list all fractions following some kind of rules? Let's have a try.

It seems like we are able to write them all row by row, but remember, our rule do not accept any kind of gaps. We should notice that start from the first row, we are not able to reach the second row no matter how hard we try. The infinity seems to block the path from the first row to the second.

But don't worry, we are still able to list all fractions through another diagnal by diagnal rule.

According to the presented figure, we can follow the 'diagnal rule' to list all fractions. For the numerators, we shall list them from 1 to k by adding one each time for each diagnal. For the denominators, we change the direction of the list and list again.





UNLISTABLE

Can we list all decimals? The answer is NO!!!!! Let's say we have a list of decimals, without consideration about its rule. If we can find a new decimal which is different from any existing element, we can prove decimals are unlistable. We also take the dignal number as the base number. Then we change each number to another one. This results in another brand new number.
